# Path planning 

This repository contains:
- A C++ implementation of the A* algorithm for path planning. 

## Prerequisites

```
	- Eigen3
	- Boost
	- OpenCV 3.4 
```

## Process (example)

### 1. Ground Annotator

Compile

```
	mkdir & cd build
	cmake ..
	make 
```

To test if it works, run the following command:
```
	cd $PATH_FINDING_DIR/build
	./astar
```
